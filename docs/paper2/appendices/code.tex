\section{Computation Script}
\label{app:code}

The following Python script reproduces all computations in
Section~\ref{sec:example}. It implements full M\"{o}bius inversion on the
capability poset, computes independent weights and $\volL(G)$, and evaluates
the three self-balancing actions. Running the script verifies every number in
the worked example.
