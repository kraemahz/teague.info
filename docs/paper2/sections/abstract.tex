\begin{abstract}
Goal-frontier maximization (GFM) aligns an agent by maximizing $\vol(G)$, the
    volume of the jointly achievable capability space \citep{lasser2026gfm}.
    The foundational paper proved structural alignment properties but left the
    capability space abstract and the measure intractable. We construct a
    computable capability space as a weighted poset discovered through agent
    communication and observation. The poset measure, defined via M\"{o}bius
    inversion, satisfies the required axioms and is polynomial-time computable.
    On this poset we define \emph{leverage}, measuring each capability's
    contribution to $\volL$ growth rate, and show that cooperation between
    high-leverage agents is disproportionately rewarded. Because $\volL$
    counts the full population's benefit, this amplification favors
    outward-facing capability-multiplying infrastructure; we argue that
    education, research, and communication are instances of this pattern,
    suggesting a structural gradient toward civilization-building implicit
    in the objective.
\end{abstract}
