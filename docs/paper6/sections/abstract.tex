\begin{abstract}
Locally rational subsumption of agents can cascade silently to full
capability domination: each step removes observation channels,
degrading the world model that would have detected the next step's
harm.  We formalize this coupled poset/world-model system and prove a
phase boundary separating self-correcting dynamics from an absorbing
basin where the world model degrades irreversibly toward monopolarity.
Cross-substrate observation-channel redundancy is the load-bearing
quantity; in two-substrate systems the boundary is narrow and the
dominant failure mode is progressive channel attenuation rather than
outright removal.  We show that three
previously disjoint alignment concerns---structural avoidance, partial
subsumption, and over-calibrated restriction---are instances of the
same dynamical system with different parameters.  The last, known as
the Wamura problem, emerges as a degenerate phase boundary with zero
self-correction rate.  The results
pair with the exogenous-verification construction
of~\citet{lasser2026exo}: that paper ensures channels are honest, this
one ensures enough honest channels survive.
\end{abstract}
